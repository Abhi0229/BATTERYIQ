\# BatteryIQ



BatteryIQ is an AI-based Battery Health \& Remaining Useful Life prediction system designed for small EV users.



\## Features

\- Capacity Retention Calculation

\- Stress Risk Classification

\- Degradation Rate Analysis

\- Remaining Useful Life Prediction

\- User-Friendly Health Report



\## Tech Stack

\- Python

\- Pandas

\- NumPy

\- Matplotlib



\## Project Structure

\- notebooks/ → Analysis \& experimentation

\- data/ → Engineered datasets

\- core/ → Core intelligence engine

\- dashboard/ → Streamlit app (future)
